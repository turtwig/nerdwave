Option Value Description
sid Which station to use. OverClocked ReMix highly recommended.
layout Where the album art goes relative to the song details.
art_size Album art size in the widget.
text_align Alignment of text.
anim_in Controls animation direction of the songs when the tracks change.
anim_out
show_duration_when_changed
show_artist Show artist or not in the widget's text.
color CSS color rule for font color. (e.g. "white"/"#FFF", "black"/"#000", or use a color picker )
background_color CSS rule for the background color for the song box. Leave blank for transparent. You may need to add your own background in OBS if using transparent.
box_shadow CSS box-shadow/glow rule for the container. (use this page for help - take what comes after "box-shadow:" and paste into this setting) Leave blank for no shadow/glow.
font_size CSS rule for font size. Can use pixels (e.g. "14px") or points. (e.g. "14pt") Leave blank for 18pt.
font_family CSS rule for what font to use. Font names with spaces must be enclosed with single quotes, e.g. 'Helvetica Neue'. Leave blank for Nerdwave defaults.
text_stroke_color CSS color rule for outlining the text. (CSS specifics: uses 4 text-shadows)
text_stroke_size CSS pixel size rule for outlining the text. (e.g. 2px) Must be in pixels or the scripting will break.
text_shadow CSS text-shadow rule. (click here for a generator) Leave blank for no shadow.
art_shadow CSS box-shadow/glow rule for the art. (use this page for a generator) Leave blank to use text_shadow, put "none" to have no shadow/glow.
width CSS rule for width of the widget (including album art, but excluding padding) before text gets clipped.
padding CSS rule for the space between the edge of the widget and the art/text. Will also be the space between the edge of the browser plugin and your OBS scene border. Leave blank for 8px.
np_header Show Current Song header.
np_message Text content of the Current Song header.
np_color Text color of the Current Song header. Default is Nerdwave's orange header color.
ad How many songs to play before showing Nerdwave's logo as the album art for a brief moment. Set to 0 to never show an ad. Ads last 2-5 seconds depending on the setting, and are Nerdwave's logo with our URL displayed in the same style as song information. (please be kind, we run this service for free :) )
ad_message Banner message for the ad. Enter in the language(s) of your stream, or leave at default for "Music by:".
delay How many seconds to wait before showing new now playing data. Make sure you use the same relay and same media player every time or you'll have to change this value constantly!
